Uploading

Before uploading your transactional data, ensure you have satisfied the following criteria:

  • Ensure the file is a CSV (.csv) file
  • Ensure the following column headers are used in your file:
    • Date
    • Amount
    • Description
  • See the example below:
Date Amount Description
2022-03-08 -100 Shopping
08/03/2022 1000 Payday
Thursday 08 March 2022 -20 Meal with friends

Balance Update

You may notice on the dashboard that there is a chart which shows how your balance has changed over time.
This is created using your current balance, and working back through all of the historical transactions.
To make sure this is correct, you'll need to enter your current balance.
Don't worry, once you've done this, any subsequent transactions will adjust your balance automatically!
